## Instalación

1. Clona este repositorio:
    ```bash
    git clone https://github.com/usuario/nombre-proyecto.git
    ```
2. Ingresa al directorio del proyecto:
    ```bash
    cd nombre-proyecto
    ```
3. Instala las dependencias:
    ```bash
    npm install
    ```

## Uso

Para iniciar el servidor:
    ``` 
    npm start
    ```
